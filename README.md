# Proyecto Huertas Urbanas – C++

Sistema en C++ para registrar huertas, analizar datos y generar reportes:
- Planta más utilizada
- Cantidad total de huertas cargadas
- Clasificación por tipo de planta
- Estadísticas generales

Tecnologías:
- C++
- Arrays
- Funciones y modularidad
